---
title: "The best long run yet"
date: 2025-04-07
---

When I went to run yesterday I was expecting a 53 minute run. But the Garmin had updated my suggested workouts overnight, so I had a 1 hour 6 minute run waiting for me.

But honestly, I was glad it did. I felt great. That’s the best feeling longer run I’ve done yet since I’ve been running these past few months. I felt strong beginning to end, and I didn’t feel like my hear rate ever spiked when I was going up any of the gradual hills around the neighborhood.

I was able to get 2 CrossFit workouts in last week in addition to all my scheduled runs, so I was excited about that. Looks like this week I’m looking at 2, maybe 3 CrossFit workouts in addition to the running. At least until I hit my June 28th 5k test, I want to focus more on the running, but I definitely want to get some CrossFit in to mix it up and get some strength work in.

The issue is with my role at work, I’ve ended up with a lot more full mornings of meetings, which makes it hard to go to CrossFit before work. I’ll figure it out.
